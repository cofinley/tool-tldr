Roles

Tool TL;DR has several roles which combine multiple permissions.

Anonymous User

  • Permissions
    • View and edit pages
    • Undo changes
  • Requirements
    • n/a
  • Conditions
    • Captchas required when editing pages with links
    • Public IP address displayed on edits

Your edits as an Anonymous user will show your public IP address to the masses. If you don't want that, you can make an account where it will become hidden forever.

User

  • Permissions
    • Anything an Anonymous User can do
    • Add pages
  • Requirements
    • Creating an account and logging in
  • Conditions
    • Captchas required when editing pages with links
  • Perks
    • No public IP displayed on edits

As a User, you get a username, you can talk about yourself in an 'about me' section, and get your public IP address hidden.

Member

  • Permissions
    • Anything a User can do
    • Move pages
  • Requirements
    • Has made 5 edits and had account for over five days
  • Perks
    • No captchas

As a Member, you can move pages around. This includes both tools and categories and serves to help organize the site's content and find the best fit for information.

Time Traveler

  • Permissions
    • Anything a Member can do
    • Moderate by easily reverting page states
  • Requirements
    • Has made 30 edits and had account for over 30 days
  • Non-Requirements
    • Roads

Right now, Time Travelers can click a button on an edit and instantly revert the page state to a previous edit. This isn't a cumulative edit nor does it destroy intermediate edits. It simply makes a new time travel edit with the same contents of the destination edit. More moderater-y permissions will be added in the future.

Administrator

  • Permissions
    • God mode
  • Requirements
    • ???
  • Conditions
    • n/a
