Change Log

v1.2

November 3, 2018
  • Allow hyperlinks to other pages
    • When editing pages, you can use an exclamation point ('!') followed by text to search for the page and insert a link to another tool or category page.
  • Fixed bug when leaving environment field blank
  • Show recent tools instead of categories on home page
  • Let users know when there are no results in search

v1.1.1

May 29th, 2018
  • Fixed bugs with the new explore tree

v1.1

May 28th, 2018
  • SAY WHAT AGAIN! Tools now have a 'what' section. Tools fell within descriptive categories, but more often than not, their descriptions deviate enough that we felt the need to let users explain them more. And, after all, it is Tool TL;DR.
  • Add 'em and Tag 'em: Environments are now tags. Tools usually aren't bound by just one type and tags look nicer anyway.
  • Tree Search: You can now filter the site tree down to find what you need. This works on the explore page as well as the 'add' and 'edit' pages.
  • Created a changelog

May 9th, 2018
  • LAUNCH!
